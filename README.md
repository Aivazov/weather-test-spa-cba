# Weather App

Невеликий пет-проект, що дозволяє отримати інформацію про погоду в обраному місті

## Як користуватися:
- У пошуковому рядку вгорі ввести назву міста (можна вибрати зі списку, що випадає)
- Додається картка з основною інформацією про погоду
- Кнопка «Видалити» видаляє картку
- Кнопка «Деталі» перенаправляє на окрему сторінку з більш детальною інформацією про погоду обраного міста
- На окремій сторінці також є можливість оновити дані про погоду обраного міста
- Перезавантаження сторінки не видаляє картки з головної сторінки

## Технології

- Next.js 16
- TypeScript
- Redux Toolkit
- RTK Query
- Material-UI
- Jest + Testing Library
- SCSS

## Встановлення та запуск

1. Клонуйте репозиторій:
```bash
git clone https://github.com/Aivazov/weather-test-spa-cba.git
cd weather-app
```

2. Встановіть залежності:
```bash
npm install
```

3. Налаштуйте змінні середовища:
Створіть файл `.env.local` в кореневому каталозі проекту:
```bash
WEATHER_API_KEY=your_openweathermap_api_key
```

Отримайте API-ключ на [OpenWeatherMap](https://openweathermap.org/api)

4. Запустіть додаток:
```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

## Тестування

```bash
npm test
```

## Розгортання на Vercel

1. Підключіть репозиторій до Vercel

2. Налаштуйте змінні середовища в Vercel:
   - `WEATHER_API_KEY` - ваш API-ключ OpenWeatherMap

3. Vercel автоматично:
   - Виконає build: `npm run build`
   - Оптимізує додаток для production
   - Налаштує CDN і маршрутизацію
